# Changelog

## [未發布] — 2026-07-19

### 新功能

- **文字貼紙（Typography Sticker v1）**：AI 生圖助手分頁化，除了原本的「圖像貼紙」流程，新增「文字貼紙」分頁。輸入一句話 → 選情緒（厭世／崩潰／熱血／治癒／文青）× 選視覺風格（粗體梗圖／漫畫標語／東方水墨／可愛貼紙）→ 產生設計感貼紙的 prompt。
- **兩種生成路線（Route A / Route B）**：Route A「圖文融合」產生單段 prompt，讓 AI 一次把字和圖畫在一起；Route B「圖字分離」拆成 Step 1 生底圖、Step 2 再加字兩段，各自可獨立複製，字比較不會翻車。選定風格時會自動帶入建議路線，使用者仍可手動切換。
- **Midjourney 使用提醒**：文字貼紙選 Midjourney 時，會提示中文字支援不佳、且無法在同一張圖續改加字（Route B 的 Step 2 請改用 ChatGPT／Gemini）。

### 改進

- **桌面版三欄式版面**：寬螢幕（≥1101px）新增 300px 右欄，AI 生圖助手整塊移到右側，左欄專心放排版設定、中間放預覽；視窗縮到 1100px 以下時自動搬回左欄原位，手機版直排行為不變。搬的是同一組 DOM 節點而非複製，所有 id 與事件綁定維持不變。
- **教學 modal 文案同步**：新手引導的說明文字跟著更新 AI 助手的新位置。

### 文件

- **架構藍圖入庫**：新增 `stickerboard architecture notes.md`，記錄雙產品線規劃（StickerBoard／未來的 LINE Helper）、Prompt Engine 的四層 prompt 結構（Text / Emotion / Aesthetic / Character）、Route A/B 設計原則，以及 Typography v1～v3 的範圍界定。

## [0.2.0] — 2026-06-27

### 新功能

- **新手引導流程**：加入逐步互動式教學提示，讓不熟悉技術的使用者也能快速上手。首次開啟時會顯示歡迎彈窗，並在操作途中提供情境式提示，降低學習門檻。
- **圓形貼紙快速套用**：新增動態圓形裁切器，一鍵把圖片套用成圓形貼紙版型。
- **AI 提示詞輔助**：整合 AI Prompt 輔助功能，讓使用者更容易描述想要的貼紙風格，生成更精準的圖像。

### 改進

- **版面完整佈局定稿**：完成貼紙排版工具的整體頁面配置，並預置好付款區塊的版位（目前為佔位符，供後續串接金流使用）。
- **畫布解析度即時同步**：畫布尺寸與解析度設定現在會即時反映到預覽，所見即所得。
- **介面重構與優化**：對 UX 操作流程進行整體重構，讓 AI 提示詞助手與畫布操作更順暢。

### 修正

- **一鍵下載 Prompt 失效**：修正「一鍵複製 AI Prompt」功能無法正常觸發的問題，現在可以順利把提示詞複製到剪貼簿。
- **初始化時版面元素消失**：修復頁面首次載入時，部分外層版面容器 (layout wrapper) 遺失，導致初始化時出現 null 樣式錯誤的問題。

### 文件

- **技術規格文件 (TECH.md)**：新增完整的技術架構文件，說明雙模式貼紙（AI 生成模式 / 手動上傳模式）的設計思路、模組結構與資料流。
- **README 更新**：補充專案說明，涵蓋功能介紹與使用方式。
