# Changelog

所有重要的版本變更都記錄在這裡。

---

## [未發布] — 2026-06-27

### 新功能

- **新手引導流程**：加入逐步互動式教學提示，讓不熟悉技術的使用者也能快速上手。首次開啟時會顯示歡迎彈窗，並在操作途中提供情境式提示，降低學習門檻。
- **圓形貼紙快速套用**：新增動態圓形裁切器，一鍵把圖片套用成圓形貼紙版型。
- **AI 提示詞輔助**：整合 AI Prompt 輔助功能，讓使用者更容易描述想要的貼紙風格，生成更精準的圖像。

### 改進

- **版面完整佈局定稿**：完成貼紙排版工具的整體頁面配置，並預置好付款區塊的版位（目前為佔位符，供後續串接金流使用）。
- **畫布解析度即時同步**：畫布尺寸與解析度設定現在會即時反映到預覽，所見即所得。
- **介面重構與優化**：對 UX 操作流程進行整體重構，讓 AI 提示詞助手與畫布操作更順暢。

### 修正

- **一鍵下載 Prompt 失效**：修正「一鍵複製 AI Prompt」功能無法正常觸發的問題，現在可以順利把提示詞複製到剪貼簿。
- **初始化時版面元素消失**：修復頁面首次載入時，部分外層版面容器 (layout wrapper) 遺失，導致初始化時出現 null 樣式錯誤的問題。

### 文件

- **技術規格文件 (TECH.md)**：新增完整的技術架構文件，說明雙模式貼紙（AI 生成模式 / 手動上傳模式）的設計思路、模組結構與資料流。
- **README 更新**：補充專案說明，涵蓋功能介紹與使用方式。
