# StickerBoard 使用指南（小白版）

> 這份是給「完全不碰程式」的人看的。照著做就能印出貼紙。
> 最後更新：2026-07-22（對應網站版本 v0.3.1）

---

## 這是什麼

一個**網頁小工具**：把你用 AI 生出來的圖片，排成一張「可以拿去印」的貼紙紙。

它不會幫你生圖，但會幫你**寫好要給 AI 的指令**（叫做 prompt，就是你貼給 ChatGPT 的那段英文），再把生好的圖排整齊、加上裁切虛線、輸出成一張圖檔。

**網址**：https://yjlhuang.github.io/stickerboard/

打開就能用，不用安裝、不用註冊、不用付錢。手機和電腦都可以。

> 怎麼確認我看到的是最新版？
> 左上角「StickerBoard」旁邊有一行小灰字，例如 `v0.3.1 · 2026-07-22`。
> 那就是版本號。跟別人討論「我這邊看到的畫面不一樣」時，先報這個數字。

## 完整流程：做一張 7-11 可以印的貼紙

### 第 1 步：想好要什麼圖，讓 AI 幫你寫指令

1. 看畫面右邊（手機是上面）的「**AI 生圖助手**」
2. 確認選在「**圖像貼紙**」這個分頁
3. 在「主題」框裡打你想要的東西，中文就行，例如：`戴眼鏡的橘貓`
4. 下面「風格」選一個（卡通插圖、水彩風、吉卜力、極簡線條、寫實風、像素風、復古印章）
5. 「AI 工具」選你等一下要用哪個 AI（通用 / ChatGPT / Gemini / Midjourney / Flux·SD）
6. 按「**📋 複製 Prompt**」

> **想做「有字的貼紙」？**（例如「還要繼續拚」這種梗圖字）
> 改點「**文字貼紙**」分頁，流程不一樣，看下面〈文字貼紙怎麼用〉。

### 第 2 步：去 AI 那邊生圖

1. 打開 ChatGPT（或你選的那個 AI）
2. 把剛剛複製的那段英文貼上去，送出
3. 等它生圖，滿意就把圖**存到電腦或手機**
4. 想要好幾種不同的貼紙，就重複幾次，存好幾張

> 助手裡會寫「**建議生圖 355px 起，最佳 1024px+**」——意思是圖太小印出來會糊。
> 用 ChatGPT / Gemini 生的圖通常都夠大，不用擔心。

### 第 3 步：回到 StickerBoard 排版

1. 左欄「**畫布**」選 `7-11 ibon 4×6"`（要用 A4 印就選 A4）
2. 左欄「**排版**」保持在「固定貼紙尺寸」，寬和高都打 `3`（＝ 3 公分的貼紙）
3. 左欄「**圖片**」區塊，把剛剛存的圖拖進去，或點一下選檔案（可以一次選很多張）
4. 中間預覽就會排好了。圖片不夠填滿？沒關係，它會自動重複使用

### 第 4 步：微調外觀（想跳過也可以）

左欄「**樣式**」區：

| 設定 | 白話解釋 |
|---|---|
| 填滿方式 | `Cover` = 填滿格子、邊邊會被裁掉；`Contain` = 整張圖都看得到、旁邊留白 |
| 貼紙內距 | 圖片離裁切線多遠。手剪容易歪的話選 2 或 3mm 比較保險 |
| 圓角半徑 | 拉到 **50%** 就變成**圓形貼紙** ⭕ |
| 格子背景色 | 預設白色。**注意**：要選透明才有用的話，你上傳的圖本身必須是去背的 |

左欄「**輸出**」區的「裁切標記」：`虛線框`（好剪）、`角落記號`（低調）、`無`。

### 第 5 步：下載並拿去印

1. 按左欄最下面的「**↓ 下載 PNG**」
2. 檔案會存到你的下載資料夾
3. 打開 **ibon 便利生活站 App** → 相片列印 → **貼紙相紙** → **4×6 英吋**
4. 上傳剛剛那張圖 → 取得代碼 → 去 7-11 的 ibon 機台印出來
5. 拿回家沿著虛線剪 ✂️

## 文字貼紙怎麼用

這是給「**想做一句話的梗圖貼紙**」用的，例如「還要繼續拚」「不想上班」。

1. AI 生圖助手 → 點「**文字貼紙**」分頁
2. 「你想說的話」打你要的那句話
3. 「**情緒**」選一個：厭世 / 崩潰 / 熱血 / 治癒 / 文青
4. 「**視覺風格**」選一個：粗體梗圖 / 漫畫標語 / 東方水墨 / 可愛貼紙
5. 「**生成方式**」會自動幫你選好，通常不用動（想知道差別看下面）
6. 「AI 工具」選你要用的 AI
7. 複製 Prompt，貼到 AI 去生圖

**生成方式的差別**（自動選好的通常就是對的）：

- **圖文融合（一步）**：一段指令，AI 一次把圖和字一起畫出來。融合度好看，適合短句、梗圖。
- **圖字分離（兩步）**：會給你 **Step 1** 和 **Step 2** 兩段。先貼 Step 1 生出「中間留白的背景圖」，滿意之後**在同一段對話裡**再貼 Step 2 叫它加字。字比較不會歪掉，適合長句和文青風。

> ⚠️ **Midjourney 的限制**：它不太會寫中文字，而且沒辦法針對同一張圖繼續改。
> 選 Midjourney 時畫面會提醒你——兩步驟的第 2 步請改用 ChatGPT 或 Gemini。

## 出事了怎麼辦

| 症狀 | 先試這個 |
|---|---|
| 按「複製 Prompt」沒反應 | 換用線上版網址（不要用本機下載的檔案開）。複製功能在某些情況下會被瀏覽器擋 |
| 印出來糊糊的 | 生圖時圖太小了。回去用 1024px 以上重生，別用截圖或縮圖 |
| 選了透明背景但還是白的 | 你上傳的圖沒有去背。先用 remove.bg 之類的免費工具去背再上傳 |
| 貼紙被裁掉一部分 | 填滿方式改成 `Contain`，或把「貼紙內距」調大 |
| 畫面跟這份指南不一樣 | 看左上角版本號。不是 `v0.3.1` 以上就是瀏覽器讀到舊的，重新整理（手機可試關掉分頁重開） |
| 歡迎彈窗一直跳 | v0.3.1 已經修好了，關掉一次之後就不會再跳。還是會跳的話代表你看到的是舊版 |
